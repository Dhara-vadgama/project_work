# 🚀 Frontend Task - React Application

## 📌 Overview

This is a frontend application built using **React (Vite)**.
The project includes authentication, protected routes, and product management features using API integration.

---

## ✨ Features

* 🔐 **Login Authentication**
* 🔒 **Protected Routes (Dashboard, Products, Profile)**
* 👤 **User Profile Page**
* 📦 **Product Listing with Pagination**
* 📄 **Product Detail Page**
* 🚪 **Logout Functionality**
* ⚡ Smooth UI with proper validation and error handling

---

## 🛠️ Tech Stack

* React.js
* Vite
* React Router DOM
* JavaScript (ES6+)
* CSS

## ⚙️ Installation & Setup

1. Clone the repository:

```bash
git clone https://github.com/Dhara-vadgama/project_work.git
```

2. Navigate to project folder:

```bash
cd project_work
```

3. Install dependencies:

```bash
npm install
```

4. Run the project:

```bash
npm run dev
```

---

## 🏗️ Build for Production

```bash
npm run build
```
